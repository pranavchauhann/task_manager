# Task Management App

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Create, view, update, delete tasks
- RESTful API
- Persistent storage

## Setup Instructions
1. Clone repository
2. Run `npm install` inside backend
3. Start MongoDB
4. Run `node server.js`
5. Open frontend/index.html

## API Endpoints
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
